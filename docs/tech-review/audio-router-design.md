# 선택적 오디오 라우터 설계 문서

본 문서는 WebRTC SFU 위에서 동작하는 선택적 오디오 라우팅 미들웨어의 설계를 다룬다. 학부 캡스톤 범위에서는 구현 외이며, 카풀링 내부 개발팀의 통합 작업 시 참고 자료로 작성되었다.

## 모듈 목적

기본 SFU 구조에서는 한 세션 내 모든 참여자가 동일한 미디어 스트림을 받는다. 그러나 카풀링의 다음 시나리오에서는 차별적 라우팅이 필요하다.

1. **골드챗 1:1 전환:** 50,000원 이상 결제한 멘티만 멘토의 비공개 음성 채널 수신
2. **음소거 멘티:** 특정 멘티의 음성을 다른 멘티들에게 전달하지 않음
3. **귓속말 기능:** 멘토 → 특정 멘티에게만 음성 메시지 전송
4. **품질 등급:** 멘티의 결제 등급에 따라 음성 비트레이트 차등 전송
